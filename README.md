# UCSB-DSCollab
Introduction to Python and Data for beginners

First workshop meeting for Data Science Collaborative Club. 

~ Worked on importing data from Kaggle
~ Organized the dataset originally from the National Institute of Diabetes and Digestive and Kidney Disease into a clear, comprehensible data table
~ The objective of the data table is to predict whether or not a patient has diabetes using diagnositc measures included in the dataset. Constraints were placed on these instances from a larger database
~ All patients shown are females who are at least 21 years of age and of Pima Indian heritage
